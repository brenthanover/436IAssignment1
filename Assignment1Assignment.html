What are the functions of HTML, CSS, and JavaScript? Well basically, HTML is the "nouns", CSS is the "adjectives", and JavaScript is the "verbs". If you only have nouns in a sentence, it's pretty clunky and hard to convey information properly.
That's why you add "adjectives". CSS adds functionality by making things easier to read, aligned, with different fonts, colours, borders, etc. This is the exact same HTML code as the previous paragraph, but with CSS styling.
Then, JavaScript adds the "verbs" to a website. JavaScript makes more complicated sites work. With HTML and CSS you can make a website look like anything, but it can only do so much to add functionality. With Javascript you can add programming to a website. The purpose of this assignment is to introduce the student to the basics of HTML, CSS, and JavaScript. The assignment must:
  1. Use at least one file of each language type
  2. Include a navbar for navigating between pages
  3. A form to add text to the page, and a button to clear
  4. A stringified JSON object that holds messages
  5. A list that contain the parsed text
  6. A button to clear the list of messages
  7. Adequate CSS styling, colors, positioning, sizing etc
  8. Something cool and extra!
